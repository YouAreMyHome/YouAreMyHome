## Giới thiệu
Xin chào! Mình là **Lê Trọng Nghĩa** — định hướng phát triển trở thành **Backend Developer**.

Mình tập trung vào việc xây dựng **API**, **hệ thống server**, **tối ưu hiệu năng**, và thiết kế **kiến trúc backend** rõ ràng, dễ mở rộng.  
Mục tiêu của mình là tạo ra các sản phẩm ổn định, bảo mật và có khả năng scale tốt.

**Mình đang quan tâm:**
- Xây dựng RESTful API / (dần mở rộng) GraphQL
- Thiết kế database (SQL/NoSQL), tối ưu truy vấn
- Authentication/Authorization (JWT, session, RBAC…)
- Caching, Queue, Background jobs
- Clean Architecture, best practices, testing
